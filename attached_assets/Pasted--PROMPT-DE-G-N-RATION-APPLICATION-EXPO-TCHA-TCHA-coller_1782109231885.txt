# PROMPT DE GÉNÉRATION — APPLICATION EXPO TCHA-TCHA
> À coller directement dans Bolt, Lovable ou Cursor pour générer l'app complète.

---

## CONTEXTE PRODUIT

Construis une application mobile complète appelée **Tcha-Tcha** avec **React Native / Expo**. C'est un outil SaaS B2B offline-first destiné aux agents Mobile Money en Afrique de l'Ouest (zone UEMOA). L'app tourne sans internet et synchronise les données quand la connexion est disponible. Le prix est 1 000 FCFA/agent/mois.

---

## STACK TECHNIQUE

- **Expo SDK** (dernière version stable) + **Expo Router** (file-based navigation)
- **TypeScript** strict
- **Supabase** (auth, base de données, realtime)
- **Zustand** (état global)
- **FedaPay** (paiement abonnement via WebView)
- **expo-sqlite** ou **MMKV** pour le stockage offline
- **react-native-reanimated** pour les animations
- **@shopify/flash-list** pour les listes de transactions
- **lucide-react-native** pour TOUS les icônes (pas de stickers, pas d'images décoratives — uniquement les icônes Lucide)
- **react-native-chart-kit** pour les graphiques

## ABONNEMENT FEDAPAY

### Écran Abonnement
- Déclenché si abonnement expiré ou depuis Profil
- Card plan : `1 000 FCFA / agent / mois`
- Calcul dynamique : nb agents × 1 000 FCFA = total
- Bouton `Payer maintenant` → ouvre FedaPay dans une WebView
- Après paiement réussi : écran succès, abonnement renouvelé 30 jours

## OFFLINE-FIRST

- Toutes les transactions sont d'abord enregistrées localement (expo-sqlite)
- File d'attente de sync (queue) : chaque transaction a un statut `synced` / `pending` / `error`
- Indicateur de sync dans le header : point vert = tout synced, orange = sync en attente, rouge = hors ligne
- Sync automatique au retour de connexion (NetInfo listener)
- Résolution de conflits : local gagne pour les nouvelles entrées, serveur gagne pour les suppressions

## NOTES IMPORTANTES

- **Aucun sticker, aucune image décorative** — uniquement des icônes `lucide-react-native`
- Tous les montants affichés sont formatés avec espaces : `50 000 FCFA`
- Le PIN est toujours masqué, affiché comme 6 cercles
- Les formulaires ont de la validation en temps réel (Zod ou validation manuelle)
- Messages d'erreur en français, langage simple et direct
- L'app est entièrement en **français**
- Toutes les dates sont au format français (`Lundi 22 juin 2026`)
- Le numéro de téléphone utilise le format Bénin (+229) par défaut, sélectionnable
- L'app doit fonctionner sur iOS et Android
